# Telegram Bot: SDXL Realism & Wan2.1 Video Generator

Этот проект представляет собой Telegram-бота на базе `aiogram`, который позволяет пользователям генерировать реалистичные фотографии по текстовому запросу и затем "оживлять" их, превращая в видеоролики.

Бот использует локальную модель Stable Diffusion XL для генерации изображений и Wan2.1 (14B) для генерации видео из изображений (Image-to-Video).

## Функционал

1. **Text-to-Image (T2I):** Пользователь отправляет текстовый запрос (prompt), и бот генерирует вертикальное изображение (896x1152) с акцентом на фотореализм.
2. **Image-to-Video (I2V):** По нажатию кнопки бот берет сгенерированное фото и создает 5-секундное видео (81 кадр, 16 FPS) с гиперреалистичным движением.

## Системные требования

- **ОС:** Linux / Windows
- **GPU:** NVIDIA GPU с поддержкой CUDA (рекомендуется от 16GB+ VRAM из-за тяжелой модели Wan2.1 14B).
- **Python:** 3.10 или выше.
